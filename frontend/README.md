# Airbnb Capstone Project

This is my Airbnb capstone project.

Users can view places, search for a location, open a listing and make a booking.

Users must sign up or log in before they can make a booking.

The admin can add, edit and delete listings. The admin can also view reservations.

## Built With

- React
- Node.js
- Express
- MongoDB
- Mongoose
- CSS
- JWT
